# Images Directory

## Hero Background Image
Please add your hero background image here and name it `hero-background.jpg` (or update the CSS to use your image filename).

The image should be:
- High resolution (1920x1080 or larger recommended)
- Optimized for web (compressed but good quality)
- Format: JPG, PNG, or WebP

If you have the image with a different name, update `LandingPage.css` line 9 to match your filename.
